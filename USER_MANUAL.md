## 1. 시스템 개요

### 1.1 시스템 소개

영업 관리 시스템은 B2B 영업 활동을 효율적으로 관리하기 위한 통합 플랫폼입니다.

**주요 기능:**

- 고객 정보 관리 및 등급 평가
- 일정(미팅, 견적, 납품) 관리
- 이메일 통합 관리
- AI 기반 고객 분석 및 추천
- 선결제 관리
- 영업 성과 분석

### 1.2 접속 방법

- **URL**: https://web-production-5096.up.railway.app
- **로그인**: 회사에서 발급받은 계정 정보 사용

## 2. 로그인 및 초기 설정

### 2.1 로그인

1. 시스템 URL 접속
2. 사용자명과 비밀번호 입력
3. "로그인" 버튼 클릭

### 2.2 프로필 확인

- 우측 상단의 사용자 이름 클릭
- 내 정보 확인:
  - 소속 회사
  - 권한 (실무자/관리자)
  - AI 기능 사용 권한

### 2.3 초기 화면 구성

로그인 후 대시보드가 표시됩니다:

- **상단**: 네비게이션 메뉴
- **좌측**: 사이드바 메뉴
- **중앙**: 주요 통계 및 작업 목록
- **우측 상단**: 사용자 정보 및 로그아웃

## 3. 대시보드 사용법

### 3.1 대시보드 구성

대시보드는 영업 활동의 전체 현황을 한눈에 확인할 수 있는 페이지입니다.

**주요 통계 카드:**

- **팔로우업**: 현재 진행 중인 고객 수
- **일정**: 오늘/이번 주 예정된 일정
- **영업 실적**: 월간/분기별 매출
- **히스토리**: 최근 활동 기록

### 3.2 빠른 작업

대시보드에서 바로 실행 가능한 작업:

- "새 고객 추가" 버튼
- "일정 등록" 버튼
- "히스토리 작성" 버튼

### 3.3 최근 활동 확인

- 오늘의 일정
- 최근 7일간 활동
- 긴급 팔로우업 고객
- 미완료 작업

## 4. 고객 관리

### 4.1 고객 목록 보기

**경로**: 좌측 메뉴 > "팔로우업 관리"

**목록 화면 구성:**

- 고객명, 회사, 부서
- 담당자 정보 (책임자, 연락처)
- 고객 등급 (VIP/A/B/C/D)
- 우선순위 (긴급/팔로업/예정)
- 상태 (진행중/완료/일시중지)

### 4.2 고객 검색 및 필터링

**검색 기능:**

- 검색창에 고객명, 회사명, 담당자명 입력
- 실시간 검색 결과 표시

**필터 옵션:**

- 고객 등급별 필터 (VIP, A, B, C, D)
- 상태별 필터 (진행중, 완료, 일시중지)
- 우선순위별 필터 (긴급, 팔로업, 예정)

**사용 예시:**

```
1. 검색창에 "서울대학교" 입력 → 서울대 관련 고객만 표시
2. 등급 "VIP" 선택 → VIP 고객만 필터링
3. 상태 "진행중" 선택 → 활성 고객만 표시
```

### 4.3 신규 고객 등록

**등록 절차:**

1. **"새 팔로우업 추가" 버튼 클릭**

   - 팔로우업 목록 상단의 파란색 버튼

2. **필수 정보 입력**

   - **고객명**: 담당자 이름 (예: 김철수)
   - **회사/학교**: 고객 소속 기관
     - 기존 회사 선택 또는 새로 추가
   - **부서/연구실**: 세부 부서 정보
     - 회사 선택 후 부서 선택/추가

3. **추가 정보 입력**

   - **책임자**: 의사결정권자 이름
     - 교수/연구원 구분: 고객명 = 책임자명 → 교수, 다르면 → 연구원
     - 대표/실무자 구분: 고객명 = 책임자명 → 대표, 다르면 → 실무자
   - **핸드폰**: 연락 가능한 전화번호
   - **이메일**: 업무용 이메일 주소
   - **주소**: 상세 주소 (택배 발송용)

4. **상태 설정**

   - **상태**: 진행중 (기본값)
   - **우선순위**:
     - 긴급: 즉시 연락 필요
     - 팔로업: 정기 관리 필요
     - 예정: 향후 연락 예정
   - **고객 등급**: AI가 자동 산정 (수동 변경 가능)

5. **메모 작성**

   - 상세 내용 필드에 특이사항 기록
   - 예: "3월 예산 집행 예정", "경쟁사와 비교 중"

6. **저장**
   - "저장" 버튼 클릭

### 4.4 고객 정보 수정

1. 고객 목록에서 수정할 고객 클릭
2. 고객 상세 페이지에서 "수정" 버튼 클릭
3. 정보 수정 후 "저장"

**수정 가능 항목:**

- 기본 정보 (이름, 연락처, 주소)
- 상태 및 우선순위
- 고객 등급 (AI 추천 확인 후 변경)
- 메모

### 4.5 고객 등급 시스템

**등급 기준:**

- **VIP**: 최우수 고객 (연 5천만원 이상 거래)
- **A**: 우수 고객 (연 2천만원 이상)
- **B**: 일반 고객 (정기 거래)
- **C**: 잠재 고객 (견적 단계)
- **D**: 저관심 고객 (장기 미접촉)

**AI 등급 업데이트:**

1. 고객 상세 페이지에서 "AI 등급 업데이트" 버튼
2. AI가 분석한 등급 및 근거 확인
3. 적용 또는 무시 선택

**전체 고객 등급 새로고침:**

- 팔로우업 목록 상단 "전체 고객 등급 새로고침" 버튼
- 마지막 갱신 이후 활동이 있는 고객만 선별하여 업데이트
- 백그라운드에서 실행되며 진행률 확인 가능

### 4.6 고객 상세 정보 확인

**탭 구성:**

- **기본 정보**: 연락처, 주소 등
- **일정**: 관련된 모든 일정
- **히스토리**: 활동 기록
- **이메일**: 주고받은 이메일
- **선결제**: 선결제 내역
- **통계**: 거래 통계 및 그래프

**빠른 작업:**

- 일정 등록
- 이메일 전송
- 히스토리 작성
- 선결제 등록

## 5. 일정 관리

### 5.1 일정 유형

시스템에서 관리하는 3가지 일정 유형:

1. **고객 미팅 (customer_meeting)**

   - 고객과의 대면/온라인 미팅
   - 상담, 제품 소개, 기술 지원 등

2. **견적 (quote)**

   - 고객에게 제공한 견적서
   - 제품, 수량, 금액 포함

3. **납품 (delivery)**
   - 실제 제품 납품 일정
   - 제품, 수량, 금액, 세금계산서 정보

### 5.2 일정 등록

#### 5.2.1 고객 미팅 등록

1. **일정 등록 화면 진입**

   - 좌측 메뉴 > "일정 관리" > "새 일정 추가"
   - 또는 고객 상세 > "일정 등록"

2. **기본 정보 입력**

   - **고객**: 미팅 대상 고객 선택
   - **일정 유형**: "고객 미팅" 선택
   - **방문 날짜**: 미팅 예정일
   - **방문 시간**: 미팅 예정 시간 (선택)

3. **미팅 상세 입력**

   - **장소**: 미팅 장소 (예: 고객 연구실, 카페, 온라인)
   - **메모**: 미팅 목적 및 준비사항
     ```
     예시:
     - 신규 장비 소개
     - 견적 상담
     - 기술 지원
     - 클레임 처리
     ```

4. **저장**

#### 5.2.2 견적 등록

1. **견적 일정 생성**

   - 일정 유형: "견적" 선택
   - 고객, 날짜 입력

2. **견적 항목 추가**

   - "납품/견적 항목 추가" 버튼 클릭
   - 제품 선택 또는 신규 제품 추가
   - 수량 입력
   - 단가 입력 (자동으로 제품 표준가 표시)
   - 할인율 적용 (선택)
   - 총액 자동 계산

3. **견적서 작성 팁**

   - **제품 코드**: 정확한 제품 선택
   - **수량**: 단위 명확히 (개, 박스, 세트 등)
   - **단가**: 할인가 반영
   - **메모**: 납기, 배송비, 특약사항 기재

4. **견적서 전송**
   - 저장 후 이메일 발송 또는 인쇄

#### 5.2.3 납품 등록

1. **납품 일정 생성**

   - 일정 유형: "납품" 선택
   - 기존 견적을 납품으로 전환 또는 직접 등록

2. **납품 항목 입력**

   - 견적 항목 복사 또는 새로 입력
   - 실제 납품 수량 확인
   - 금액 확정

3. **세금계산서 정보**

   - **공급가액**: 납품 금액 (VAT 별도)
   - **세액**: 공급가액의 10%
   - **합계금액**: 공급가액 + 세액
   - **발행일**: 세금계산서 발행 예정일

4. **납품 완료 처리**
   - 상태를 "완료"로 변경
   - 실제 납품일 기록

### 5.3 일정 조회

#### 5.3.1 목록 보기

**일정 관리 페이지**

- 전체 일정 리스트 형태로 표시
- 날짜순 정렬

**필터링:**

- 일정 유형별 (미팅/견적/납품)
- 상태별 (예정/진행중/완료/취소)
- 기간별 (오늘/이번 주/이번 달/기간 지정)

#### 5.3.2 캘린더 보기

**달력 뷰**

- 월별 캘린더에 일정 표시
- 색상 구분:
  - 🔵 파란색: 고객 미팅
  - 🟢 초록색: 견적
  - 🔴 빨간색: 납품

**사용법:**

- 날짜 클릭 → 해당일 일정 상세
- 일정 클릭 → 일정 상세 페이지
- 드래그 앤 드롭으로 일정 이동 (개발 예정)

### 5.4 일정 수정 및 관리

**일정 수정:**

1. 일정 클릭 → 상세 페이지
2. "수정" 버튼
3. 정보 수정 후 저장

**상태 변경:**

- **예정 → 진행중**: 일정 시작 시
- **진행중 → 완료**: 일정 완료 시
- **취소**: 일정 취소 시 (사유 기재)

**일정 삭제:**

- 삭제 버튼 클릭
- 확인 메시지 확인 후 삭제
- ⚠️ 주의: 삭제 시 복구 불가

## 6. 히스토리 관리

### 6.1 히스토리란?

히스토리는 고객과의 모든 접촉 활동을 기록하는 로그입니다.

**히스토리 vs 일정:**

- **일정**: 미래 또는 예정된 활동
- **히스토리**: 이미 발생한 활동의 기록

**히스토리 종류:**

- 전화 통화
- 이메일 송수신
- 미팅 결과
- 견적 제공
- 납품 완료
- 기타 활동

### 6.2 히스토리 작성

**작성 경로:**

- 좌측 메뉴 > "히스토리 관리" > "새 히스토리 추가"
- 고객 상세 페이지 > "히스토리 작성"

**작성 항목:**

1. **기본 정보**

   - **고객**: 대상 고객 선택
   - **활동 유형**:
     - 전화 통화
     - 이메일
     - 고객 미팅
     - 견적 제공
     - 납품 완료
     - 기타

2. **활동 내용 (중요!)**

   - 상세하고 구체적으로 작성
   - AI 분석에 활용됨

   **좋은 예시:**

   ```
   - 신규 장비 구매 문의. 예산 2천만원 내외.
   - 3월 중 발주 예정. 경쟁사 2곳과 비교 중.
   - 다음 주 화요일 재방문 약속.
   ```

   **나쁜 예시:**

   ```
   - 전화함
   - 미팅했음
   ```

3. **날짜/시간**

   - 활동 발생 날짜 (기본: 오늘)
   - 시간 (선택)

4. **첨부파일**

   - 관련 문서, 사진 등 첨부 가능

5. **저장**

### 6.3 히스토리 조회 및 검색

**조회 옵션:**

- 전체 히스토리 목록
- 고객별 히스토리
- 기간별 필터링
- 활동 유형별 필터링

**검색:**

- 내용으로 검색
- 고객명으로 검색

### 6.4 히스토리 활용

**AI 분석용 데이터:**

- 히스토리 내용은 AI 고객 분석에 활용
- 고객 관심사, 구매 패턴 파악
- 팔로우업 우선순위 산정

**보고서 작성:**

- 고객 보고서에 히스토리 자동 포함
- 활동 실적 집계

## 7. 이메일 관리

### 7.1 Gmail 연동

**연동 방법:**

1. 우측 상단 사용자 메뉴 > "Gmail 연동"
2. Google 계정으로 로그인
3. 권한 승인
4. 연동 완료 확인

**연동 후 기능:**

- 시스템 내에서 이메일 발송
- 수신 이메일 자동 수집
- 고객별 이메일 스레드 관리

### 7.2 이메일 작성 및 발송

#### 7.2.1 일반 이메일 작성

1. **작성 화면 열기**

   - 좌측 메뉴 > "이메일 관리" > "새 이메일"

2. **수신자 입력**

   - 고객 선택 (자동으로 이메일 주소 채움)
   - 또는 직접 이메일 주소 입력

3. **제목 및 본문 작성**

   - 제목 입력
   - 본문 작성 (HTML 에디터 제공)

4. **발송**
   - "전송" 버튼 클릭

#### 7.2.2 AI 이메일 작성

**AI 자동 생성 기능:**

1. 이메일 작성 화면에서 "AI 이메일 생성" 클릭
2. 조건 입력:
   - **목적**: 신규 작성 or 답장
   - **톤**: 격식체 / 일반체 / 간결체
   - **내용 요약**: 전달할 내용 간단히 입력
     ```
     예시:
     - 신규 장비 소개 및 견적 제안
     - 미팅 일정 조율
     - 납품 일정 안내
     ```
3. "생성" 버튼
4. AI가 생성한 이메일 확인
5. 필요시 수정 후 전송

**AI 이메일 변환:**

- 작성한 초안을 다른 톤으로 변환
- 예: 격식체 → 일반체

### 7.3 이메일 수신 및 관리

**수신 이메일 확인:**

- 이메일 관리 페이지에서 수신함 확인
- 고객별로 스레드 형태로 표시

**답장하기:**

1. 이메일 클릭
2. "답장" 버튼
3. 원문이 자동으로 인용됨
4. 답장 작성 및 전송

**이메일-고객 연결:**

- 자동: 발신자 이메일로 고객 자동 매칭
- 수동: 이메일 상세에서 고객 직접 연결

## 8. 제품 관리

### 8.1 제품 등록

**등록 경로:**

- 좌측 메뉴 > "제품 관리" > "새 제품 추가"

**입력 항목:**

1. **제품 코드** (필수)

   - 고유한 제품 식별 코드
   - 예: "ABC-001", "LAB-EQUIP-2024-01"

2. **제품 설명** (필수)

   - 제품명 및 상세 설명
   - 예: "원심분리기 3000rpm 모델"

3. **표준 가격**

   - 정가 (할인 전 가격)
   - 숫자만 입력 (쉼표 자동 추가)

4. **프로모션 가격** (선택)

   - 할인가 또는 특별가
   - 입력 시 목록에서 강조 표시

5. **재고 수량** (선택)

   - 현재 재고량

6. **활성화 상태**

   - ✅ 활성: 판매 가능
   - ❌ 비활성: 단종 또는 임시 중단

7. **저장**

### 8.2 제품 조회

**제품 목록:**

- 회사별 제품만 표시 (타 회사 제품 비표시)
- 제품 코드, 설명, 가격 표시
- 견적 횟수 및 판매 횟수 통계

**검색:**

- 제품 코드 또는 설명으로 검색

**필터:**

- 활성/비활성 제품 필터

**정렬:**

- 제품 코드순
- 가격순
- 견적 횟수순
- 판매 횟수순

### 8.3 제품 수정 및 삭제

**수정:**

1. 제품 목록에서 제품 클릭
2. "수정" 버튼
3. 정보 수정 후 저장

**가격 변경 시 주의:**

- 기존 견적/납품의 가격은 변경되지 않음
- 새 견적부터 변경된 가격 적용

**삭제:**

- 삭제 버튼 클릭
- ⚠️ 주의: 이미 사용된 제품은 삭제 불가 (비활성화 권장)

### 8.4 제품 활용

**견적/납품 시 제품 선택:**

- 제품 선택 드롭다운에 자동 표시
- 표준가 자동 입력
- 수량 입력하면 총액 자동 계산

**제품 통계:**

- 제품별 견적 제공 횟수
- 제품별 실제 판매 횟수
- 제품별 매출액

## 9. 선결제 관리

### 9.1 선결제란?

고객이 미리 입금한 금액을 관리하는 기능입니다.

**사용 사례:**

- 정기 거래 고객의 사전 입금
- 대량 구매 계약금
- 연간 계약 선지급

### 9.2 선결제 등록

**등록 절차:**

1. **등록 화면 진입**

   - 좌측 메뉴 > "선결제 관리" > "새 선결제 추가"
   - 또는 고객 상세 > "선결제 등록"

2. **기본 정보 입력**

   - **고객**: 선결제한 고객 선택
   - **회사**: 고객 소속 회사 (자동 입력)

3. **입금 정보**

   - **입금액**: 실제 입금된 금액
   - **입금 날짜**: 입금일
   - **입금자명**: 입금한 사람 이름
   - **입금 방법**: 계좌이체/현금/카드 등

4. **메모**

   - 선결제 사유 또는 특이사항
   - 예: "2025년 1분기 구매 계약금"

5. **저장**

### 9.3 선결제 사용 (차감)

**납품 시 자동 차감:**

1. 납품 일정 등록 시
2. 해당 고객의 선결제 잔액 확인
3. 납품 금액만큼 자동 차감

**수동 차감:**

1. 선결제 상세 페이지
2. "사용 내역 추가"
3. 사용 금액 및 사유 입력

### 9.4 선결제 조회

**잔액 확인:**

- 선결제 목록에서 현재 잔액 표시
- 고객 상세 페이지에도 표시

**사용 내역:**

- 선결제 상세 > "사용 내역" 탭
- 날짜, 금액, 사유 표시

**만료/취소:**

- 상태를 "취소" 또는 "만료"로 변경
- 취소 사유 필수 입력

## 10. AI 기능 활용

### 10.1 AI 권한 확인

- AI 기능은 관리자가 부여한 사용자만 이용 가능
- 우측 상단 프로필에서 "AI 권한" 확인

### 10.2 AI 고객 등급 평가

**개별 고객 등급 업데이트:**

1. 고객 상세 페이지
2. "AI 등급 업데이트" 버튼
3. AI 분석 결과 확인:
   - 추천 등급 (VIP/A/B/C/D)
   - 점수 (0-100)
   - 평가 근거
   - 주요 요인
   - 개선 제안
4. "적용" 또는 "무시" 선택

**전체 고객 등급 새로고침:**

1. 팔로우업 목록 상단
2. "전체 고객 등급 새로고침" 버튼
3. 업데이트 정보 확인:
   - 마지막 갱신 시간
   - 처리 대상 고객 수 (변경된 고객만)
   - 예상 소요 시간
4. 백그라운드 실행
5. 진행률 확인 (모달 창)
6. 완료 후 변경 내역 확인

**효율적 운영:**

- 첫 실행: 모든 활성 고객 평가
- 이후 실행: 마지막 갱신 이후 활동이 있는 고객만 평가
- 토큰 절약 및 시간 단축

### 10.3 AI 팔로우업 우선순위 제안

**사용 방법:**

1. 팔로우업 목록 상단
2. "AI 우선순위 제안" 버튼
3. AI 분석 시작 (최대 50명)
4. 우선순위 결과 확인:
   - 고객명 및 회사
   - 고객 구분 (교수/연구원/대표/실무자)
   - 우선순위 점수
   - 우선순위 등급 (긴급/높음/중간/낮음)
   - 제안 이유
   - 제안 행동
   - 최적 연락 시간

**AI가 고려하는 요소:**

- 마지막 연락 경과 시간
- 진행 중인 영업 기회
- 구매 패턴 및 재구매 시기
- 고객 등급
- 선결제 잔액
- 견적 후 미구매 기간
- 고객 구분 (의사결정권자 우선)
- 최근 히스토리 메모 내용

**결과 활용:**

- 오늘 연락할 고객 선정
- 우선순위별 업무 계획 수립

### 10.4 AI 이메일 생성

**기능:**

- 이메일 자동 작성
- 답장 자동 생성
- 톤 변환 (격식체/일반체/간결체)

**사용법:**

1. 이메일 작성 화면
2. "AI 생성" 버튼
3. 조건 입력:
   - 목적 (신규/답장)
   - 톤 선택
   - 제품/서비스 정보
   - 전달 내용 요약
4. 생성된 이메일 확인 및 수정
5. 전송

**팁:**

- 내용 요약을 구체적으로 작성할수록 정확한 결과
- 생성 후 꼭 검토 및 수정
- 고객별 맞춤 수정 권장

### 10.5 AI 고객 요약

**고객 프로필 자동 생성:**

1. 고객 상세 페이지
2. "AI 요약 생성" 버튼
3. AI가 분석한 요약 확인:
   - 고객 개요
   - 주요 관심사
   - 거래 이력 요약
   - 최근 활동
   - 특이사항

**활용:**

- 신규 담당자 인수인계
- 미팅 전 고객 정보 파악
- 보고서 작성

### 10.6 AI 제품 추천

**맞춤형 제품 추천:**

1. 고객 상세 페이지
2. "AI 제품 추천" 버튼
3. AI가 분석한 추천 제품:
   - 추천 제품 목록
   - 추천 이유
   - 예상 관심도

**추천 근거:**

- 과거 구매 이력
- 업종 및 연구 분야
- 유사 고객 구매 패턴

## 12. 문제 해결

### 12.1 자주 묻는 질문 (FAQ)

**Q1: 로그인이 안 됩니다.**

- A: 비밀번호 확인, 관리자에게 계정 확인 요청

**Q2: AI 기능이 보이지 않습니다.**

- A: 관리자가 AI 권한을 부여했는지 확인

**Q3: 다른 회사 제품이 보입니다.**

- A: 시스템 오류 가능성, 관리자에게 문의

**Q4: 이메일 발송이 안 됩니다.**

- A: Gmail 연동 상태 확인, 재연동 시도

**Q5: 일정이 캘린더에 안 보입니다.**

- A: 페이지 새로고침, 필터 설정 확인

### 12.2 오류 발생 시

**일반적인 해결 방법:**

1. 페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 다른 브라우저로 시도
4. 로그아웃 후 재로그인

**데이터 손실 방지:**

- 중요한 작업 전 저장
- 긴 텍스트는 별도 저장 후 복사

### 12.3 고객 지원

**문의 방법:**

- 시스템 관리자에게 이메일 또는 메신저
- 오류 메시지 스크린샷 첨부
- 발생 시점 및 재현 방법 설명

**긴급 상황:**

- 데이터 손실: 즉시 관리자 연락
- 시스템 다운: 백업 시스템 사용 (문의)
